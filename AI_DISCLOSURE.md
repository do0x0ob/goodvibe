# AI 使用揭露

本專案依 Sui Vibe Hackathon 規定揭露 AI 使用情況。**整份專案都是 vibe coding，我沒有自己打過任何一行代碼**, This is my vibe 😄

## 我用了哪些 AI

| 模型 | 主要用途 |
|------|----------|
| **Claude Sonnet 4.5** | Sui 相關的建置：Move 合約、鏈上邏輯、交易與查詢 |
| **Gemini 3 Pro** | 前端與排版設計、UI 與版面 |
| **Claude Opus** | 解 Stable Layer SDK 跟 Next.js 16 Turbo Pack Buffer 的衝突、複雜整合問題 |
| **Cursor Auto** | CLI 腳本、以及可以直接指名「改這裡、做那個」的具體修改 |
| **Cleancode skill** | 自訂的 AI 重構原則，讓 AI 照著做程式碼整理與重構 |

另外在 Cursor 裡也會用到 Composer / Agent 做日常的程式撰寫、除錯、註解與文件。

## 聲明

產品方向與要什麼功能是我決定的；實作則全部透過與上述 AI 協作完成。
